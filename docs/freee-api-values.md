# freee API 定数・値リファレンス

## 消費税関連

### tax_entry_method (内税・外税区分)

| 値    | 説明 |
| ----- | ---- |
| `in`  | 内税 |
| `out` | 外税 |

### tax_fraction (消費税端数計算)

| 値         | 説明     |
| ---------- | -------- |
| `omit`     | 切り捨て |
| `round`    | 四捨五入 |
| `round_up` | 切り上げ |

### tax_rate (税率)

| 値   | 説明         |
| ---- | ------------ |
| `10` | 標準税率 10% |
| `8`  | 軽減税率 8%  |
| `0`  | 非課税・免税 |

## 源泉徴収

### withholding_tax_entry_method (源泉徴収計算方法)

| 値    | 説明             |
| ----- | ---------------- |
| `in`  | 税込金額から計算 |
| `out` | 税抜金額から計算 |

※ `none` は API スキーマでは無効。源泉徴収なしの場合は `out` を使用し、明細行の `withholding: false` で対応。

## 敬称

### partner_title

| 値         | 説明                  |
| ---------- | --------------------- |
| `御中`     | 法人向け              |
| `様`       | 個人向け              |
| `(空白)`   | 敬称なし (半角カッコ) |
| `（空白）` | 敬称なし (全角カッコ) |

## 支払方法

### payment_type

| 値             | 説明             |
| -------------- | ---------------- |
| `transfer`     | 振込             |
| `direct_debit` | 振替（口座引落） |

## 明細行タイプ

### lines.type

| 値     | 説明                   |
| ------ | ---------------------- |
| `item` | 品目行（通常の明細）   |
| `text` | テキスト行（金額なし） |

## よく使う単位

| 値     | 用途     |
| ------ | -------- |
| `式`   | 一式     |
| `個`   | 個数     |
| `本`   | 本数     |
| `時間` | 時間単価 |
| `人月` | 人月単価 |
| `ヶ月` | 月額     |

## 請求書ステータス

| ステータス         | 説明     |
| ------------------ | -------- |
| `draft`            | 下書き   |
| `issue`            | 発行済み |
| `sending`          | 送付中   |
| `sent`             | 送付済み |
| `payment_received` | 入金済み |

## 税区分コード (tax_code) - よく使うもの

| コード | 説明               |
| ------ | ------------------ |
| `21`   | 課税売上10%        |
| `23`   | 課税売上8%（軽減） |
| `0`    | 対象外             |
| `1`    | 非課税売上         |

## APIエンドポイント

### ベースURL

| API            | URL                                              |
| -------------- | ------------------------------------------------ |
| freee請求書API | `https://api.freee.co.jp/iv`                     |
| freee会計API   | `https://api.freee.co.jp/api/1`                  |
| OAuth認証      | `https://accounts.secure.freee.co.jp/public_api` |

### 請求書API エンドポイント

| メソッド | パス                  | 説明             |
| -------- | --------------------- | ---------------- |
| GET      | `/invoices`           | 一覧取得         |
| GET      | `/invoices/{id}`      | 個別取得         |
| POST     | `/invoices`           | 作成             |
| PUT      | `/invoices/{id}`      | 更新             |
| GET      | `/invoices/templates` | テンプレート一覧 |

### 会計API エンドポイント (補助)

| メソッド | パス             | 説明         |
| -------- | ---------------- | ------------ |
| GET      | `/companies`     | 事業所一覧   |
| GET      | `/partners`      | 取引先一覧   |
| GET      | `/account_items` | 勘定科目一覧 |
| GET      | `/taxes/codes`   | 税区分一覧   |
| GET      | `/items`         | 品目一覧     |
| GET      | `/sections`      | 部門一覧     |

## HTTPステータスコード

| コード | 説明           |
| ------ | -------------- |
| `200`  | 成功           |
| `201`  | 作成成功       |
| `400`  | リクエスト不正 |
| `401`  | 認証エラー     |
| `403`  | 権限エラー     |
| `404`  | リソース不在   |
| `429`  | レート制限超過 |
| `500`  | サーバーエラー |
