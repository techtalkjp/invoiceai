# 画面設計: 月次経費請求

## 変更対象画面

### 1. クライアント詳細 > 経費項目タブ（新規）

経費グループと項目の管理画面。

```
┌─ 経費項目 ─────────────────────────────────┐
│                                              │
│  [+ グループ追加]  [+ 項目追加]               │
│                                              │
│  ▼ サーバ通信費  (USD / 3項目 / $45.00/月)   │
│    │  税率 10%                               │
│    ├── Vercel           fixed   $20.00       │
│    ├── Supabase         fixed   $15.00       │
│    └── Resend           fixed   $10.00       │
│                                              │
│  ▼ AI API利用料  (USD / 1項目 / 従量)        │
│    │  税率 10%                               │
│    └── Gemini API       metered  GCP         │
│        └── dailove-search / Cloud AI API     │
│                                              │
│  端数処理: [四捨五入 ▼]                       │
│                                              │
└──────────────────────────────────────────────┘
```

項目の追加/編集ダイアログ:

- 共通: 名前、種別（fixed/metered）、有効期間（開始月〜終了月、任意）
- fixed: 月額（USD）
- metered: プロバイダ選択 → プロバイダ固有設定（GCPの場合: プロジェクトID、サービスフィルタ）

### 2. 請求書作成画面の変更

既存の稼働時間明細に加え、経費明細セクションを追加。

```
┌─ 請求書プレビュー ──────────────────────────┐
│                                              │
│  稼働明細                                    │
│  ┌──────────────────┬────┬────────┬────────┐ │
│  │ 摘要             │数量│ 単価   │ 金額   │ │
│  ├──────────────────┼────┼────────┼────────┤ │
│  │ 開発・保守 3月分  │16h │ 20,000│320,000 │ │
│  └──────────────────┴────┴────────┴────────┘ │
│                                              │
│  経費明細                                    │
│  為替: 1 USD = [149.53] 円 (日銀TTM 3/31)    │
│  ※手動変更可  [レート取得]                    │
│  ┌──────────────────┬────┬────────┬────────┐ │
│  │ 摘要             │数量│ 単価   │ 金額   │ │
│  ├──────────────────┼────┼────────┼────────┤ │
│  │ サーバ通信費 3月  │  1 │  6,729│  6,729 │ │
│  │ Gemini API 3月   │  1 │      9│      9 │ │
│  └──────────────────┴────┴────────┴────────┘ │
│                                              │
│  [経費再取得] — 従量課金の最新値を取り直す     │
│                                              │
│  ℹ GCPの3月利用データは暫定値の可能性があります│
│  （対象月の翌月5日より前の場合に表示）          │
│                                              │
└──────────────────────────────────────────────┘
```

- 為替レートは自動取得。手動で上書き可能（上書き時は理由入力を求める）
- 従量課金は「経費再取得」ボタンで最新値を取り直せる
- 経費明細の金額は自動計算（外貨 × レート → 端数処理）
- **確定**: 請求書を作成（freee送信）した時点で `invoice_line` に凍結
